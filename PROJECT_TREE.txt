DATA MODELS
===========

Report Object:
{
  reportId: "RPT-{timestamp}-{random}",
  timestamp: "ISO 8601 string",
  location: {
    latitude: Number,
    longitude: Number,
    governorate: String,
    region: String
  },
  description: String,
  photos: {
    incident: "filename.jpg",
    face: "filename.jpg"
  }
}

Governorate Object:
{
  id: Number,
  name_en: String,
  name_ar: String,
  regions: [String]
}
